==============================================
Aqserver - Siemens S7 data acquisition server
==============================================
The program is an acquisition tool for Siemens Simatic S7 PLCs.
It was written using the python-snap7 module from https://pypi.python.org/pypi/python-snap7/ and the snap7 library from http://snap7.sourceforge.net .
File handling is based on example source code for the book "Real-World Instrumentation with Python".
